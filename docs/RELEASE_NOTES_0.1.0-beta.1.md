# 555 Stream 0.1.0-beta.1

First public preview of the canonical `555 Stream` plugin.

## Included

- canonical package metadata and schema-driven plugin surface
- public ecosystem docs linkage
- action and operator documentation
- public-preview skills for operators and OpenClaw-style agents
- CI validation and release-check coverage

## Public preview scope

- auth and session bootstrap
- channel configuration and go-live control
- ads, overlays, chat, guests, media, and studio state

## Known limitations

- not GA
- compositor/ad parity and some live-status accuracy remain active product gaps
- see `docs/COVERAGE_AND_GAPS.md` for current truth
